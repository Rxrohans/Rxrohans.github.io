DSO Graph WebApp

Streamlit Tool to Plot Oscilloscope Signals

Overview

This project is a lightweight tool to plot Digital Storage Oscilloscope (DSO) signal data using a Streamlit web app.

Objectives

  • Upload unstructured DSO `.csv` data
  • Visualize voltage-time waveforms
  • Enable signal analysis for electronics debugging

Skills Used

  • Python, Pandas
  • Streamlit
  • Matplotlib
